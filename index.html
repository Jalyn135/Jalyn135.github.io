💪 Weekly Health Tracker

Java | Object-Oriented Programming | Data Management

A practical health and wellness application that helps users track their weekly meal intake against personalized calorie goals. Demonstrates OOP principles with multiple classes working together seamlessly.

Features:

  • Set personalized weekly calorie goals
  • Browse curated list of 16 healthy meal options
  • Log meals throughout the week
  • Automatic calorie calculation and weekly summary
  • Goal comparison feedback (over, under, or exact match)

Classes:

  • Meal.java - Represents individual meal data
  • HealthTracker.java - Manages weekly meal collection and calculations
  • Main.java - User interface and program flow
